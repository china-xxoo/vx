配置页面

这个页面用来生成主页面 index.html 里面的 DEFAULT_ENCRYPTED_CONFIG 和 ADMIN_PASSWORD_HASH。

注意：这里输入的 GitHub Token、聊天密钥、万能管理密码只用于生成加密结果。 生成后复制到 index.html。不要把 Token 明文写进公开代码。
聊天加密密钥用于加密房间信息、房间列表和聊天内容。你和对方使用同一套网页时，这个密钥必须一致。
推荐暗号：123456✖️
在计算器主页面输入：1 → 2 → 3 → 4 → 5 → 6 → × → =
万能管理密码用于房间里的 房间设置。输入这个密码后，可以删除房间和该房间聊天记录。 生成后，主页面代码里只保存它的 SHA-256 哈希，不保存明文密码。
配置已生成，可以复制到 index.html 里。
复制后，到 index.html 里搜索 DEFAULT_ENCRYPTED_CONFIG 和 ADMIN_PASSWORD_HASH，把对应内容替换掉。
